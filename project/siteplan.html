Website Site Plan

Site Name

Site Name: Ijeoma Web Dev

This site name was chosen because the website documents my personal growth as a web development student. It clearly communicates that the site is a learning portfolio showcasing my progress, skills, and projects throughout the WDD 131: Dynamic Web Fundamentals course.

Optional Domain: ijwebdev.com

Site Purpose

The purpose of this website is to showcase my learning journey in web development as a student in WDD 131. The site will introduce who I am, highlight the HTML, CSS, and JavaScript skills I have acquired, display selected course projects, and provide helpful learning resources. The site will also include a contact or feedback form with JavaScript validation to demonstrate dynamic functionality.

Scenarios

  • Who is the student behind this web development portfolio?
  • What web development skills has the student learned so far?
  • Can I view examples of projects built using HTML, CSS, and JavaScript?
  • Where can I find useful resources for learning web development?

Color Schema

  • Primary Color: #1FA3E0

    Used for buttons, branding, and major visual highlights.

  • Active/Secondary Color: #F3A712

    Used for active navigation links and interactive emphasis.

  • Background Colors: #060B10 to #0F2A33

    A dark radial gradient used to provide depth and modern contrast.

  • Text Color: #FFFFFF (Main) and #E5F1F7 (Paragraphs)

    Chosen for high readability against the dark background.

Typography

  • Heading Font: Poppins, sans-serif – Used for headings to create a clean and professional look.
  • Body Font: Open Sans, sans-serif – Used for body text, lists, and form content for readability.
